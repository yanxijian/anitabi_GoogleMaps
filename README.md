## 使用说明

0. 脚本需要 Python 运行环境，并依赖 `requests` 库（安装： `pip install requests`）
1. 打开网站 [https://www.anitabi.cn/map](https://www.anitabi.cn/map)，搜索你的作品获取所需圣地巡礼地图的 bangumiId
2. 下载并运行脚本，根据提示输入 bangumiId，按回车开始下载地图信息，下载完后会保存到当前目录，文件名是 `{地图标题名}.kml`
3. 打开 [https://www.google.com/maps/d/home?hl=zh-CN](https://www.google.com/maps/d/home?hl=zh-CN)，选择创建新地图，在新图层（如果没有则手动添加）上传导入文件 `{地图标题名}.kml`
